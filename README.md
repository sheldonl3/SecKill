## golang秒杀后台系统
### 项目优势
1.代码和部署完全隔离，避免相互影响<br>
2.模块化，解耦<br>
3.根据后端承载能力，进行限流<br>
4.使用Redis承载海量QPS<br>
5.有过载保护和降级预案<br>
6.平台化，所有操作在web平台实时生效<br>
